# 🍽️ Foodie's Restaurant Website

A modern, responsive, and visually appealing restaurant website built using **HTML5, CSS3, and JavaScript**. This project showcases a clean user interface, smooth navigation, interactive sections, and a mobile-friendly design, making it ideal for learning front-end web development and building a professional portfolio.

## 🚀 Features

- 🍔 Modern Hero Section
- 📱 Fully Responsive Design
- 🧭 Sticky Navigation Bar
- 🏡 About Us Section
- 🍕 Popular Menu Cards
- 👨‍🍳 Meet Our Chef Section
- 📸 Food Gallery
- ⭐ Customer Testimonials
- 📅 Table Reservation Form
- 📞 Contact Section
- 🎨 Smooth Hover Effects
- ⬆️ Back to Top Button
- ✨ Scroll Reveal Animations

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
